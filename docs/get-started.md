# Get started

This part of the documentation helps you find the best way to get started with Nick, depending on what you want to do.

## Try a demo

You can try out a demo of Nick combined with the Volto frontend at [https://demo.nickcms.org](https://demo.nickcms.org)

## Install Nick

To get started with Nick you can use the [installation guide](/admin-guide/installation) to setup your own Nick installation.
